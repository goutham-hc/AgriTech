# 🌾 AgriTech — AI-Powered Smart Farming Platform

A machine learning web application built for Karnataka farmers.

## Features
- 🌱 Crop Recommendation (89.7% accuracy)
- 📊 Yield Prediction (Low/Medium/High)
- 💰 Crop Price Prediction
- 🍃 Plant Disease Detection (Coming Soon)

## Tech Stack
- Python, Flask, Scikit-learn
- HTML, CSS, JavaScript
- Random Forest ML Models
- Karnataka Crop Dataset (2004-2019)
